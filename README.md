# Həkim Borc Dəftəri

Həkimlər üçün xəstə borc və nisyə idarəetmə proqramı.

## Xüsusiyyətlər

- **Xəstə Qeydiyyatı** - Xəstə məlumatlarını əlavə edin, redaktə edin və axtarın
- **Borc/Nisyə İzləmə** - Əməliyyatları qeyd edin, borcları izləyin
- **Ödəniş Qeydləri** - Tam və qismən ödənişləri qeyd edin (nağd, kart, köçürmə)
- **Hesabatlar** - Aylıq/illik gəlir hesabatları, qrafiklər
- **Xatırlatmalar** - Gecikmiş ödənişlər üçün xatırlatmalar

## Quraşdırma

1. Setup faylını işə salın
2. Quraşdırma təlimatlarını izləyin
3. Proqramı açın və istifadəyə başlayın

## Sistem Tələbləri

- Windows 10 və ya daha yuxarı
- 4 GB RAM (minimum)
- 500 MB boş disk sahəsi

## Dəstək

Suallarınız üçün əlaqə saxlayın.
